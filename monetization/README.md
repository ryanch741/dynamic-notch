# Dynamic Notch 商业化配置指南

## 平台选择

### 1. LemonSqueezy（推荐）
- **优势**：专门为软件销售设计，支持订阅、许可证管理
- **费用**：3% + $0.30 per transaction
- **特点**：内置 DRM、自动更新机制

### 2. Gumroad
- **优势**：老牌平台，用户基数大
- **费用**：5% + $0.30 per transaction
- **特点**：易于设置，营销工具丰富

### 3. Paddle
- **优势**：专注于软件销售，欧洲合规
- **费用**：5% per transaction
- **特点**：税务处理完善，支持多种货币

## 产品设置步骤

### 基础信息
- **产品名称**: Dynamic Notch
- **标语**: A macOS utility for Dynamic Island-style interaction on notch-equipped MacBook Pros
- **描述**: 详细的产品功能介绍，突出5大核心功能
- **价格建议**: $9.99 USD (一次性购买) 或 $2.99 USD/月 (订阅)

### 产品变体
1. **免费版** - 仅包含基础功能（时间日期、快捷方式）
2. **专业版** - 包含所有功能（番茄钟、音乐控制、待办事项）
3. **终身版** - 一次性购买，永久使用所有功能

### 许可证验证
- 集成许可证验证SDK
- 支持离线验证
- 定期激活检查

## 集成步骤

### LemonSqueezy 集成示例

1. **创建商店**：
   - 访问 https://lemonsqueezy.com
   - 注册账户并创建新产品

2. **配置产品**：
   ```json
   {
     "product_name": "Dynamic Notch Professional",
     "price": 9.99,
     "currency": "USD",
     "download_url": "https://your-app.com/download",
     "license_validation": true
   }
   ```

3. **集成 SDK**：
   - 使用 LSWrapper 或类似的 Swift 包装库
   - 实现许可证验证逻辑
   - 添加购买页面的深度链接

### 功能限制策略
- **免费版**：仅限3个快捷方式，基础时间显示
- **专业版**：解锁所有功能，无数量限制
- **企业版**：团队许可证，批量折扣
