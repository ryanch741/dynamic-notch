## Product Hunt 发布文案

### 标语 (Tagline)
> 专为 macOS 刘海屏设计的效率增强工具

### 描述
Dynamic Notch 是一款为 MacBook Pro 刘海屏设计的创新效率工具。当你的鼠标靠近刘海区域时，它会智能展开，展示一系列实用功能模块。

### 核心亮点
- **智能交互**: 鼠标悬停自动展开，平滑动画体验
- **多功能集成**: 时间日期、快捷方式、番茄钟、音乐控制、待办事项
- **多显示器支持**: 可在副屏显示模拟刘海条
- **国际化支持**: 中英文界面自动切换
- **隐私安全**: 本地运行，无云端数据传输

### 适用人群
- 经常使用 MacBook Pro 刘海屏的用户
- 需要提高工作效率的专业人士
- 喜欢 macOS 原生体验的技术爱好者
- 专注力管理需求的用户
