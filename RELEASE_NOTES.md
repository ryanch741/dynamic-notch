# Notch Island (灵动刘海) v1.0.0 正式版发布

🎉 欢迎使用“灵动刘海” v1.0.0 正式版！这是一个专为 macOS 刘海屏设计的效率增强工具。

### ✨ 核心特性
- **动态刘海交互**：鼠标靠近时自动展开，平滑的缩放动画。
- **效率模块集成**：
    - **时间日期**：实时显示精确时间与星期。
    - **快捷方式**：一键开启常用应用、网站和文件夹。
    - **番茄钟**：专注力管理，支持系统级通知提醒。
    - **音乐控制**：实时显示 Apple Music / Spotify 播放信息并支持切歌。
- **个性化定制**：
    - 适配物理刘海位置。
    - 自定义“黑洞”主题图标。
    - 支持开机自启动配置。

### 📦 安装说明
1. 下载 `NotchIsland-1.0.0.dmg`。
2. 双击打开并将“灵动刘海”拖拽到 `Applications` (应用程序) 文件夹。
3. 首次启动请在“系统设置 -> 隐私与安全性”中允许“辅助功能”权限，以便应用能正常监听鼠标位置并控制音乐。
