# 功能更新日志

## v1.0.1 - 开机自启动功能

### 新增功能
- **开机自启动** - 应用现在支持在系统启动时自动运行
  - 在设置界面添加了"开机自启动"开关
  - 支持中英文界面显示
  - 使用 macOS 标准的登录项管理

### 技术实现
- 创建了 `LaunchAtStartupManager.swift` 服务类
- 集成到现有设置界面
- 添加了相应的本地化字符串

### 文件变更
- `NotchIslandApp.swift` - 添加了 ServiceManagement 框架和 LaunchAtStartupManager
- `SettingsView.swift` - 添加开机自启动开关
- `Localizable.strings` - 添加中英文翻译
- `Info.plist` - 添加必要配置

### 测试要点
- [ ] 开机自启动开关功能正常
- [ ] 设置界面显示正确
- [ ] 多语言支持正常
- [ ] 应用正常启动和退出

## v1.0.0 - 初始发布版本

### 核心功能
- **动态刘海交互** - 鼠标靠近时自动展开，平滑的缩放动画
- **时间日期模块** - 实时显示精确时间与星期
- **快捷方式模块** - 一键开启常用应用、网站和文件夹
- **番茄钟模块** - 专注力管理，支持系统级通知提醒
- **音乐控制模块** - 实时显示 Apple Music / Spotify 播放信息并支持切歌
- **待办事项模块** - 便捷的 Todo 列表管理，集成在菜单栏
- **多显示器支持** - 可选在副屏上显示模拟刘海条
- **国际化支持** - 根据系统语言自动切换中英文界面

### 发布内容
- macOS DMG 安装包
- GitHub Release
- Homebrew Cask 配置
- 双语文档支持
